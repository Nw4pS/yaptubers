# Introduction
Yet Another PNG Tuber Software is a simple software that displays a default image and changes it if the microphone detects high enough volume from the microphone

| not talking                  | talking                        |
| ---------------------------- | ------------------------------ |
| ![silentImage](./silent.png) | ![talkingImage](./talking.png) |

# usage
In order to change the shown images, you have to put them in the directory of the executable and rename them "silent.png" and "talking.png"
